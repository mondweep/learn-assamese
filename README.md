# Learn Assamese - Language Learning Platform

A modern, AI-powered language learning platform designed to help Assamese diaspora communities preserve their cultural heritage and maintain connection to their roots.

## üéØ Mission

Assamese diaspora communities are growing worldwide, creating a cultural disconnect between generations. This platform provides a reliable, engaging, mainstream resource for learning Assamese language and culture, addressing the lack of such tools in existing platforms like Duolingo.

## üöÄ Getting Started (For Contributors)

### Prerequisites
- Node.js >= 16.0.0
- npm >= 7.0.0
- API keys (Anthropic or Google Gemini for research-swarm)

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd learn-assamese

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env and add your API keys

# Initialize research-swarm database
npx research-swarm init
```

### Using Research-Swarm

research-swarm is integrated for ongoing market and technical research:

```bash
# Example: Research a specific topic
npx research-swarm goal-research "Your research question here" --depth 5

# See all available commands
npx research-swarm --help
```

**Full research guide:** [RESEARCH_GUIDE.md](./RESEARCH_GUIDE.md)
